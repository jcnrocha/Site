Cláusulas fundamentais do SELECT
SELECT
Define quais colunas retornar. Use * para selecionar todas.
SELECT titulo, ano
FROM
Especifica de qual tabela os dados serão buscados.
FROM filme
WHERE
Aplica uma condição de filtro — retorna apenas as linhas que atendem ao critério.
WHERE ano >= 2020
ORDER BY
Ordena o resultado. ASC crescente (padrão) e DESC decrescente.
ORDER BY ano DESC
DISTINCT
Elimina linhas duplicadas no resultado — retorna cada valor único uma só vez.
SELECT DISTINCT genero

⏱️ Duração: 15 minutos
👥 Agrupamento: turma inteira · resposta oral individual
🖊️ Materiais: slides com os comandos · quadro branco

Desenvolver a leitura e interpretação de comandos SELECT em linguagem natural, fixando o papel de cada cláusula por meio da tradução do código SQL para o português.

1
Exibição do comando (1 min por rodada)
Professor projeta um comando SELECT sobre o banco da locadora. A turma deve "traduzir" o que o comando faz em uma frase em português.
2
Tradução oral (1 min)
Professor escolhe um aluno para traduzir o comando. A turma complementa ou corrige a resposta.
3
Consolidação (1 min)
Professor confirma a tradução correta, destaca a cláusula trabalhada e mostra o resultado esperado da consulta.
RodadaComando SQLTradução esperada
1 SELECT titulo, ano FROM filme; "Mostre o título e o ano de todos os filmes cadastrados"
2 SELECT * FROM cliente
WHERE cidade_cliente = 'Curitiba';		 "Mostre todos os dados dos clientes que moram em Curitiba"
3 SELECT titulo FROM filme
ORDER BY ano DESC;		 "Mostre os títulos dos filmes ordenados do mais recente para o mais antigo"
4 SELECT DISTINCT genero FROM filme; "Mostre cada gênero de filme uma única vez, sem repetição"
5 SELECT titulo FROM filme
WHERE ano >= 2020 AND genero = 'Ação';		 "Mostre o título dos filmes de ação lançados a partir de 2020"
📌 Por que usar o banco da locadora? A turma já conhece esse banco desde a SA 15 — elimina a necessidade de apresentar um novo contexto e permite que o foco fique 100% na leitura e interpretação do SELECT.

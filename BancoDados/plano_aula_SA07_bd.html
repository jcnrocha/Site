📚 Conteúdo da Aula Ferramentas · Conexão · Mercado
GUI × CLI — Duas Formas de Interagir com o Banco
GUI
Interface Gráfica
  • Interface visual com menus, botões e formulários
  • Visualização de tabelas, dados e estrutura do banco
  • Ideal para desenvolvedores e DBAs no dia a dia
  • Facilita modelagem, diagramas e exportação de dados
  • Exemplos: MySQL Workbench, DBeaver, pgAdmin
CLI
Linha de Comando
  • Interação por comandos digitados no terminal
  • Essencial em servidores sem interface gráfica
  • Permite automação via scripts e agendamentos
  • Mais rápido para tarefas repetitivas e em lote
  • Exemplos: mysql, psql, mongosh
Principais Ferramentas do Mercado
🐬
MySQL Workbench
MySQL GUI Oficial
  • Modelagem visual e diagramas ER
  • Editor SQL com syntax highlighting
  • Administração do servidor MySQL
  • Importação e exportação de dados
🐘
pgAdmin
PostgreSQL Web / GUI
  • Interface web rodando no navegador
  • Gerenciamento de objetos do banco
  • Editor de queries com plano de execução
  • Monitoramento de atividade e conexões
🌿
MongoDB Compass
MongoDB GUI Oficial
  • Visualização e edição de documentos JSON
  • Editor de queries e agregações
  • Análise de índices e performance
  • Exportação de dados em CSV e JSON
🦫
DBeaver
Multi-SGBD Gratuita
  • Suporte a mais de 80 SGBDs diferentes
  • Editor SQL avançado com autocomplete
  • Visualização e edição de dados em grade
  • Preferida por desenvolvedores no dia a dia
📋
TablePlus
Multi-SGBD Freemium
  • Interface moderna e intuitiva
  • Conexão rápida com múltiplos bancos
  • Edição inline de registros
  • Versão gratuita com funcionalidades limitadas
📦
DB Browser for SQLite
SQLite Gratuita
  • Leve e simples — ideal para iniciantes
  • Abre arquivos .db e .sqlite diretamente
  • Sem necessidade de servidor
  • Muito usada em projetos mobile e desktop
Parâmetros de Conexão — Portas Padrão dos SGBDs
SGBD Porta padrão Host local Usuário padrão Observação
MySQL 3306 127.0.0.1 root Requer servidor MySQL em execução
PostgreSQL 5432 127.0.0.1 postgres Requer servidor PostgreSQL em execução
MongoDB 27017 127.0.0.1 — Sem autenticação por padrão no desenvolvimento
SQLite Arquivo local — — Sem servidor — conecta diretamente ao arquivo .db
💡 Para o desenvolvedor: ao configurar uma conexão em qualquer ferramenta GUI, os campos necessários são sempre os mesmos — host (endereço do servidor), porta, usuário, senha e nome do banco. O SQLite é a exceção: basta apontar para o arquivo.

🎮 Dinâmica Quem Sou Eu? — Round Ferramentas
🔍 Quem Sou Eu? — Identificação por Pistas
⏱️ Duração: 15 minutos
👥 Agrupamento: grupos de 4 a 5 alunos
🏆 Pontuação: 3 pts na 1ª pista · 2 pts na 2ª · 1 pt na 3ª
1
Leitura das pistas (1 min/rodada) — Professor lê as pistas em ordem crescente de especificidade. Grupos tentam identificar a ferramenta após cada pista.
2
Resposta em grupo (1 min) — O grupo que identificar primeiro levanta a mão. Pontuação conforme a pista em que acertou.
3
Consolidação (1 min) — Professor confirma a resposta e reforça as características da ferramenta identificada.
# Pista 1 3 pts Pista 2 2 pts Pista 3 1 pt Resposta
1 Sou uma ferramenta gráfica oficial de um dos SGBDs mais usados no mundo Permito modelar visualmente o banco de dados e criar diagramas ER Sou da Oracle e meu nome tem a ver com o banco que gerencio e com uma bancada de trabalho MySQL Workbench
2 Sou gratuita e consigo me conectar a quase qualquer banco de dados do mercado Tenho suporte a mais de 80 SGBDs diferentes em uma única interface Meu nome vem de um personagem dos desenhos animados e meu logo é um castor DBeaver
3 Sou a ferramenta oficial de um banco de dados que guarda documentos JSON Permito visualizar, filtrar e editar documentos diretamente pela interface Meu nome tem a ver com a palavra "bússola" e sou feita pela mesma empresa do meu SGBD MongoDB Compass
4 Sou leve, simples e não preciso de servidor para funcionar Sou ideal para projetos de aplicativos mobile que precisam de um banco local Meu nome tem a ver com navegar em um banco de dados que cabe em um arquivo DB Browser for SQLite
5 Sou a ferramenta oficial de administração de um banco de dados open source muito robusto Funciono como uma aplicação web que roda no navegador Meu nome começa com "pg" — a sigla do banco de dados que gerencio pgAdmin
💡 Ao final da dinâmica, projetar a tabela resumo com as 5 ferramentas, seus SGBDs correspondentes e casos de uso — reforçando a associação que será necessária nas próximas aulas práticas.

Pontos Críticos para a Prova Trimestral

🔌
Bloco A — Infra, Conexões e IPs
  • Cabeamento e Sinais: Fibra multimodo para curtas distâncias corporativas e monomodo para longas distâncias (800m+). Wi-Fi 2.4GHz (maior alcance, menor velocidade) vs 5GHz.
  • Conectividade: FTTH (fibra na residência), VDSL, 5G para IoT e baixa latência (~1ms), satélites LEO (órbita baixa/baixa latência) para áreas rurais.
  • IP e NAT: IPs privados (faixas RFC 1918) para redes locais; NAT faz a tradução de vários privados para um único público da internet. IPv6 com 128 bits.
Dica: Pratique a identificação de conexões ideais para diferentes perfis de clientes/localizações.
🛡️
Bloco B — Protocolos, OSI e Segurança
  • TCP vs UDP: TCP garante a integridade de dados (handshake, confirmações e retransmissões); UDP foca em rapidez (streaming, jogos, VoIP, DNS).
  • DNS, DHCP & HTTPS: DNS traduz nomes em IPs; DHCP atribui IPs por meio do processo DORA. HTTPS utiliza TLS/SSL para cifrar dados de navegação.
  • Segurança & OSI: Firewalls controlam tráfego, VPNs criam túneis criptografados. O Modelo OSI de 7 camadas serve de referência para diagnósticos lógicos.
Dica: Use o macete "Fui Embora, Reginaldo! Tô Sem Amor, Afinal" para ordenar as 7 camadas OSI.

  • B) Cabo coaxial
  • C) Fibra óptica monomodo (✓ Resposta Correta)

2. Um dispositivo recebe automaticamente o endereço IP 192.168.1.45 ao se conectar a uma rede Wi-Fi. Qual protocolo realizou essa configuração?
  • A) DNS
  • B) FTP
  • C) HTTP
  • D) DHCP (✓ Resposta Correta)
3. Qual protocolo de transporte é mais indicado para uma videochamada ou streaming em tempo real?
  • A) TCP — pela entrega garantida de todos os pacotes
  • B) UDP — pela latência mínima e velocidade de envio (✓ Resposta Correta)
  • C) FTP — pela velocidade de sincronização de arquivos
  • D) HTTP — por se tratar de um protocolo padrão da web

  • B) Camada 2 — Enlace de Dados

  • B) DDoS
  • C) Phishing (✓ Resposta Correta)
  • D) Man-in-the-Middle
Questões Discursivas do Simulado
Q1. Explique a diferença entre endereço IP privado e endereço IP público. Dê um exemplo de cada.
Resposta Esperada: O IP privado é de uso interno nas redes locais (ex: 192.168.1.10) e não navega diretamente na internet. O IP público é o endereço visível e roteável na internet atribuído ao roteador de borda pelo provedor (ex: 177.45.200.10). O NAT é o responsável por traduzir IPs privados em público para permitir a navegação.
Q2. O que é a tríade CIA e qual a sua importância prática para a segurança de sistemas em redes?
Resposta Esperada: Trata-se dos três pilares da segurança: Confidencialidade (acesso apenas a pessoas permitidas), Integridade (garantia de não-alteração indevida de dados) e Disponibilidade (serviços e redes funcionais). É a base que orienta todas as políticas, softwares e configurações de segurança da informação em redes.
Q3. Descreva o funcionamento do processo DORA do protocolo DHCP.
Resposta Esperada: Discover: o cliente envia broadcast procurando um servidor DHCP. Offer: o servidor oferece uma configuração de IP. Request: o cliente solicita formalmente usar o IP oferecido. Acknowledge (ACK): o servidor confirma e concede a utilização do IP temporariamente (tempo de lease).

## Objetivos de Aprendizagem

Ao final da aula, o aluno será capaz de:

- Retomar e consolidar os conceitos de componentes e ativos de rede (H3) trabalhados no 1º trimestre como base para o 2º
- Compreender como os conteúdos já vistos se conectam com as tecnologias de conexão que serão estudadas no 2º trimestre
- Reconhecer os conceitos introdutórios de meios de transmissão — preparando o terreno para a Aula 14
- Identificar as questões que surgem naturalmente após conhecer os componentes de rede: *"como os dados efetivamente viajam por esses cabos e dispositivos?"*
- Criar motivação e curiosidade para os conteúdos do 2º trimestre a partir da conexão com situações reais do mercado de TI

**Habilidade da matriz:** H3 — Reconhecer componentes e ativos de redes · H4 — Identificar tipos e tecnologias de conexão a redes de computadores

## Conteúdo

- Esta aula é de transição — não há conteúdo novo extenso · o foco é retomar, conectar e preparar
- **Retomada do 1º Trimestre — o que já sabemos:**
  - **H1:** bits, bytes, KB, MB, GB, TB · bps, Kbps, Mbps, Gbps · latência, throughput · conversão de unidades
  - **H2:** simbologias de rede — roteador, switch, AP, servidor, firewall, DMZ, nuvem · leitura e criação de diagramas
  - **H3:** componentes passivos (cabos, conectores, patch panel, rack) · componentes ativos (hub, switch, roteador, AP, modem, NIC)
  - **H4 introdução:** interfaces (Ethernet, Wi-Fi, fibra, USB) · serviços de internet (ADSL, FTTH, HFC, 4G, 5G, satélite)

- **A pergunta que abre o 2º Trimestre:**
  - *"Agora que conhecemos os componentes — os cabos, os switches, os roteadores e os servidores — como os dados efetivamente viajam por essa infraestrutura? O que determina a velocidade, a confiabilidade e a segurança dessa transmissão?"*

- **Introdução ao conceito de Meios de Transmissão:**
  - Um meio de transmissão é o caminho físico ou eletromagnético pelo qual os dados viajam de um dispositivo a outro
  - Os meios guiados (cabos) foram introduzidos na Aula 07 — agora vamos aprofundar cada tipo
  - Os meios não guiados (Wi-Fi, Bluetooth, satélite) foram mencionados na Aula 09 — agora vamos detalhar cada tecnologia
  - A escolha do meio de transmissão impacta diretamente: velocidade, distância, custo, segurança e interferência

- **Conexão H3 → H4: a pergunta natural após conhecer os componentes**

  No 1º trimestre aprendemos:
  - O switch conecta dispositivos usando o endereço MAC — mas *como ele descobre esse endereço?*
  - O roteador encaminha pacotes usando o endereço IP — mas *como o IP é atribuído? Como o roteador decide o melhor caminho?*
  - O cabo Cat6A suporta 10 Gbps — mas *como os bits viajam fisicamente pelo fio de cobre?*
  - A fibra óptica transmite dados por luz — mas *como a luz é modulada para carregar informação?*
  - O Wi-Fi conecta dispositivos sem fio — mas *quais são os padrões, as frequências e como evitar interferências?*

  Essas perguntas são exatamente o que o 2º trimestre vai responder.

## Dinâmica — "O Que Ainda Não Sei?"

**Duração:** 15 minutos · **Agrupamento:** grupos de 4 a 5 alunos

**Objetivo:** ativar a curiosidade natural dos alunos sobre os conteúdos do 2º trimestre, identificando as lacunas de conhecimento que ficaram após o 1º trimestre e transformando-as em perguntas motivadoras — mostrando que o 2º trimestre existe exatamente para responder essas questões.

**Materiais:** post-its ou tiras de papel · canetas · painel ou quadro para fixar · ou slide para registrar as perguntas digitalmente.

**Passo a passo:**

1. **Geração de perguntas** *(5 min)*
Cada grupo discute e escreve em post-its as perguntas que surgiram ao longo do 1º trimestre e que ainda não foram respondidas. Exemplos de perguntas que podem surgir:
- *"Como o roteador decide qual caminho usar para enviar um pacote?"*
- *"O que é um endereço IP e como ele é diferente do endereço MAC?"*
- *"Por que o Wi-Fi cai quando o micro-ondas liga?"*
- *"O que é DNS e como ele funciona?"*
- *"O que acontece quando dois computadores tentam enviar dados ao mesmo tempo?"*

2. **Fixação no painel** *(3 min)*
Cada grupo fixa seus post-its no quadro ou painel. O professor agrupa as perguntas por tema.

3. **Conexão com o 2º trimestre** *(7 min)*
O professor lê cada pergunta, identifica em qual aula do 2º trimestre ela será respondida e cria expectativa:
- *"Essa pergunta sobre endereço IP? Respondemos na Aula 18."*
- *"Por que o Wi-Fi cai com o micro-ondas? Aula 15 — meios não guiados e interferência."*
- *"O que é DNS? Aula 20 — protocolos de aplicação."*

**Variação para turmas mais tímidas:**
Se a turma for reservada para criar perguntas, o professor pode usar cartões com perguntas prontas e pedir que os grupos identifiquem em qual aula do mapa do 2º trimestre cada pergunta seria respondida.

**Perguntas sugeridas pelo professor (caso necessário):**

| Pergunta | Aula onde será respondida |
|----------|--------------------------|
| Como o IP de um dispositivo é atribuído automaticamente? | Aula 18 — DHCP |
| O que acontece nos bastidores quando você digita "google.com" no navegador? | Aula 20 — DNS |
| Por que a fibra óptica é mais rápida que o cabo de cobre? | Aula 14 — Meios Guiados |
| Como os dados chegam ao seu celular sem nenhum cabo? | Aula 15 — Meios Não Guiados |
| O que é o modelo OSI e para que serve? | Aula 22 — Modelo OSI |
| Como uma VPN deixa a conexão segura? | Aula 21 — Segurança Básica |
| Por que o TCP é mais confiável que o UDP? | Aula 19 — Protocolos TCP/IP e UDP |
| Como a internet chega à sua casa pela fibra da operadora? | Aula 16 — Tecnologias de Conexão Parte 1 |

> **Nota:** o objetivo desta dinâmica não é ensinar conteúdo novo — é criar expectativa, curiosidade e motivação para o 2º trimestre. Reforçar que todas as perguntas geradas serão respondidas ao longo das próximas aulas — o 2º trimestre existe exatamente para isso.

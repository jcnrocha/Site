# UC: Fundamentos de Tecnologias da Informação
## SA 07 — 1º Trimestre

---

## Tema

Sistemas Operacionais · Conceito, tipos, funções e interface

---

## Objetivos de Aprendizagem

Ao final da SA, o aluno será capaz de:

- Compreender o conceito de Sistema Operacional e sua função como intermediário entre hardware e software
- Identificar os principais sistemas operacionais do mercado — Windows, Linux, macOS, Android e iOS
- Diferenciar interface gráfica (GUI) de linha de comando (CLI) e reconhecer as situações de uso de cada uma
- Relacionar o sistema operacional ao perfil do usuário e ao contexto de uso — desktop, mobile e servidor
- Reconhecer a habilidade H6 — identificar tipos e funcionalidades dos sistemas operacionais

---

## Conteúdo

- O que é um Sistema Operacional — conceito, função e importância
- Funções do SO — gerenciar hardware, memória, processos, arquivos e usuários
- Principais SOs de desktop — Windows, Linux e macOS: características, vantagens e desvantagens
- Principais SOs mobile — Android e iOS: diferenças e market share
- SOs para servidor — Linux Server e Windows Server
- Interface gráfica (GUI) x linha de comando (CLI) — quando e por que usar cada uma
- Habilidade H6 — identificar tipos e funcionalidades dos sistemas operacionais

## Dinâmica — Qual SO Sou Eu?

**Nome:** Qual SO Sou Eu?  
**Duração:** 10 a 15 minutos  
**Agrupamento:** 5 grupos de 4 alunos

**Descrição:** O professor projeta características, logos ou descrições de sistemas operacionais sem revelar o nome. Cada grupo identifica qual SO está sendo descrito e justifica a resposta. Professor valida e reforça os conceitos.

**Passo a passo:**

1. **Apresentação** — Professor projeta a descrição ou imagem sem o nome do SO
2. **Análise em grupo** — Cada grupo tem 20 segundos para identificar o SO
3. **Resposta simultânea** — Grupos revelam a resposta ao mesmo tempo
4. **Correção** — Professor valida, revela o nome e explica as características

**Descrições sugeridas:**

| # | Descrição projetada | SO correto |
|---|---|---|
| 1 | "Código aberto, gratuito, muito usado em servidores e por desenvolvedores" | Linux |
| 2 | "Sistema da Apple exclusivo para computadores Mac, integrado ao iPhone" | macOS |
| 3 | "Mais usado no mundo em desktops, vem pré-instalado na maioria dos PCs" | Windows |
| 4 | "Sistema mobile do Google, presente em mais de 70% dos smartphones" | Android |
| 5 | "Sistema mobile da Apple, exclusivo para iPhone e iPad" | iOS |
| 6 | "Usado em empresas para gerenciar redes, usuários e serviços corporativos" | Windows Server |

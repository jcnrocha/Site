# UC: Fundamentos de Tecnologias da Informação
## SA 06 — 1º Trimestre

---

## Tema

Fundamentos de Software · Conceito, tipos, licenças e entrega do trabalho em grupo

---

## Objetivos de Aprendizagem

Ao final da SA, o aluno será capaz de:

- Compreender o conceito de software e diferenciá-lo de hardware
- Classificar os tipos de software — de sistema, aplicativo e utilitário — com exemplos do cotidiano
- Identificar os tipos de licença de software — proprietário, free, freeware, shareware e open source
- Reconhecer a função de drivers e utilitários no funcionamento do sistema computacional
- Apresentar o trabalho em grupo sobre Ferramentas Google, demonstrando o que foi pesquisado pela equipe

---

## Conteúdo

- O que é software — conceito e relação com o hardware
- Tipos de software — de sistema (SO), aplicativo e utilitário
- Licenças de software — proprietário, free, freeware, shareware e open source
- Drivers — o que são, para que servem e como funcionam
- Utilitários — antivírus, compactadores, desfragmentadores e ferramentas de backup
- Como o software é criado — breve visão do ciclo de desenvolvimento
- Entrega e apresentação do trabalho em grupo — Ferramentas Google

## Dinâmica — Que Tipo de Software Sou Eu?

**Nome:** Que Tipo de Software Sou Eu?  
**Duração:** 10 a 15 minutos  
**Agrupamento:** 5 grupos de 4 alunos

**Descrição:** O professor projeta nomes de softwares conhecidos, um por vez. Cada grupo decide em qual categoria o software se encaixa — de sistema, aplicativo ou utilitário — e qual tipo de licença possui. O grupo que acertar mais ao final vence.

**Passo a passo:**

1. **Apresentação** — Professor projeta o nome do software
2. **Análise em grupo** — Cada grupo tem 20 segundos para decidir o tipo e a licença
3. **Resposta simultânea** — Grupos revelam a resposta ao mesmo tempo
4. **Correção** — Professor valida e explica

**Softwares sugeridos:**

| Software | Tipo | Licença |
|---|---|---|
| Windows 11 | De sistema (SO) | Proprietário |
| LibreOffice | Aplicativo | Open source |
| VLC Media Player | Aplicativo | Open source |
| WinRAR | Utilitário | Shareware |
| Linux Ubuntu | De sistema (SO) | Open source |
| Microsoft Word | Aplicativo | Proprietário |
| Avast Free Antivírus | Utilitário | Freeware |
| Google Chrome | Aplicativo | Freeware |

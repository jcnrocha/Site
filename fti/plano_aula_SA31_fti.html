🏆 Objetivos de Aprendizagem SA 31

Ao final da SA, o aluno será capaz de:

  • Compreender o que é Python, por que é amplamente usado no mercado e sua relação com o desenvolvimento de sistemas
  • Configurar e utilizar o ambiente de desenvolvimento — IDLE ou VS Code — para escrever e executar código Python
  • Declarar variáveis e identificar os tipos de dados básicos — int, float, str e bool
  • Utilizar as funções print() e input() para realizar saída e entrada de dados
  • Converter tipos de dados com int(), float() e str() e realizar operações aritméticas simples

🎮 Dinâmica — Qual o Tipo? SA 31
🎮 Dinâmica — Qual o Tipo?

Nome: Qual o Tipo?

Duração: 10 a 15 minutos

Agrupamento: 5 grupos de 4 alunos

Descrição: O professor projeta trechos de código Python com variáveis e valores atribuídos. Cada grupo identifica o tipo de dado de cada variável — int, float, str ou bool — e indica se haveria erro de tipo em alguma operação apresentada.

Passo a passo:

  • 1Apresentação — Professor projeta o trecho de código
  • 2Análise em grupo — Cada grupo identifica os tipos e possíveis erros
  • 3Resposta simultânea — Grupos revelam a resposta ao mesmo tempo
  • 4Correção — Professor valida e explica o comportamento do Python

Trechos de código sugeridos:

# Código Resposta esperada
1 `nome = "João"` str
2 `idade = 17` int
3 `altura = 1.75` float
4 `aprovado = True` bool
5 `resultado = "5" + 3` ❌ Erro de tipo — não pode somar str com int
6 `resultado = int("5") + 3` ✅ int — conversão correta, resultado = 8
7 `media = (7 + 8 + 9) / 3` float — resultado = 8.0
8 `resto = 10 % 3` int — resultado = 1

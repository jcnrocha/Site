H03 — Identificar características de modelagem de dados
🏆 Objetivos de Aprendizagem 5 objetivos

Ao final da aula, o aluno será capaz de:

  • 1Compreender o que é modelagem de dados e qual é o seu papel no desenvolvimento de um sistema
  • 2Diferenciar modelo de dados de banco de dados — entendendo que o modelo é o projeto, e o banco é a implementação
  • 3Identificar as três fases da modelagem: conceitual, lógica e física, e o que cada uma representa
  • 4Reconhecer qual artefato corresponde a cada fase — DER, esquema de tabelas ou script SQL
  • 5Relacionar a necessidade de modelar com problemas reais causados por bancos mal estruturados

🎮 Dinâmica Obrigatória
🔍 Em Qual Fase Estou?
⏱️ Duração: 15 minutos
👥 Agrupamento: grupos de 3 a 4 alunos
🎴 Materiais: slides com os artefatos ou cartões impressos

Fixar a distinção entre as três fases da modelagem por meio de classificação de artefatos reais de um mesmo sistema fictício: sistema de locadora de filmes.

1
Distribuição dos artefatos (2 min)
O professor exibe ou distribui 6 artefatos — 2 de cada fase — relacionados ao sistema de locadora. Os artefatos aparecem embaralhados, sem indicação de fase.
2
Classificação em grupo (7 min)
Cada grupo classifica cada artefato em uma das três fases: Conceitual, Lógica ou Física. O grupo deve justificar em uma frase por que escolheu aquela fase.
3
Apresentação e debate (5 min)
Cada grupo apresenta sua classificação para um artefato sorteado. O professor media o debate e consolida as respostas corretas.
ArtefatoFase correta
Diagrama com retângulos "Filme" e "Cliente" ligados por losango "Aluga" Conceitual
Diagrama com entidade "Filme" com atributos (título, gênero, ano) sem tipos SQL Conceitual
Tabela escrita: FILME (id_filme PK, titulo, genero, ano) Lógica
Tabela escrita: LOCACAO (id_locacao PK, id_cliente FK, id_filme FK, data_locacao) Lógica
CREATE TABLE filme (id_filme INT PRIMARY KEY, titulo VARCHAR(150) NOT NULL, ano INT) Física
CREATE TABLE locacao (id_locacao INT PRIMARY KEY, id_cliente INT, id_filme INT, data_locacao DATE) Física
Nota ao professor: os artefatos devem ser apresentados embaralhados, sem indicação de fase. O objetivo é que os alunos identifiquem a fase pelo nível de abstração e pelo tipo de representação.
